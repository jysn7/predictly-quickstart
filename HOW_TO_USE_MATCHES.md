# How to Use Real Matches Feature

## Quick Start

### On the Predict Page
1. **View All Matches** - Page automatically loads 16 real upcoming matches
2. **Search for Teams** - Type in the search box:
   - "Manchester" to find Manchester teams
   - "Liverpool" to find Liverpool matches
   - "Real Madrid" to find Spanish teams
   - "NBA" to find basketball matches
3. **Filter by Sport** - Use the dropdown to filter:
   - "All Sports" - See everything
   - "Football" - Premier League, La Liga, Ligue 1
   - "Basketball" - NBA games
   - "Cricket" - International matches
   - "Tennis" - ATP and WTA tournaments
   - "American Football" - NFL games
   - "Ice Hockey" - NHL games

4. **Select a Match** - Click any match to select it (it will highlight with a purple border)
5. **Generate Prediction** - Click "Get Prediction" button to let AI predict the outcome

### On the Feed Page
1. **View All Matches** - See upcoming matches from other users
2. **Filter by Sport** - Select a specific sport from dropdown
3. **View Predictions** - See what other users predict for each match
4. **Click Explain** - Get AI explanation for any prediction

## Tips & Tricks

1. **Quick Search** - Type just first few letters:
   - "Man" finds Manchester teams
   - "Liv" finds Liverpool
   - "Prem" finds Premier League

2. **Case Insensitive** - Search works with any case:
   - "manchester" = "Manchester" = "MANCHESTER"

3. **Sport Filter First** - Filter by sport first, then search within that sport for faster results

4. **Date Format** - Matches show:
   - "Today at 20:00" - for today
   - "Tomorrow at 19:30" - for tomorrow
   - "Nov 15 at 14:00" - for future dates

5. **Auto-Sorting** - Results automatically sort by date (earliest first)

## Troubleshooting

### No matches appearing?
- Check that you typed the search correctly
- Try selecting "All Sports" filter
- Refresh the page

### Search not finding match?
- Search is case-insensitive, so any case works
- Search looks for partial matches (e.g., "Real" finds "Real Madrid")
- Try different keywords (team name vs league vs sport)

### Selected match won't predict?
- Make sure match is selected (should have purple border)
- Check that "Get Prediction" button is enabled
- Try refreshing page if it seems stuck
