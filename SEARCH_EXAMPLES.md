## üí° Pro Tips

1. **Search is case-insensitive** - "manchester", "Manchester", "MANCHESTER" all work
2. **Partial matches work** - "Man" finds "Manchester", "prem" finds "Premier"
3. **Try multiple searches** - If one doesn't work, try a different search term
4. **Use sport filter first** - Then search within that sport
5. **Location + Team** - Search "Old Trafford" to see Man United matches
6. **League + Day** - Search "Premier" then filter to find specific day
